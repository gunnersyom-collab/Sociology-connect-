Sociology Connect
​Ibsa Gabaabaa (Project Overview)
Sociology Connect marsariitii barattoota damee Sociology fi Social Work tajaajiluudha. Marsariitiin kun odeeffannoo waa'ee barnoota Sociology, Social Work, Iddir, fi Mahber kan of keessatti qabateedha.
​Kaayyoo (Goal)
​Barattootaaf odeeffannoo walitti hidhuu fi hawaasa sociology keessatti wal gargaaruu.
​Hojiiwwan Federalism Club kan Arsi University irratti akkaataa itti hojjetamu ifatti ibsuu.
​Teeknooloojii Fayyadame (Tech Stack)
​HTML: Ijaarsa fi qabiyyee websaayitichaa uumuuf fayyadame.
​CSS: Dizaayinii fi bifa websaayitichaatiif.
​Python: Ijaarsa ammayyaa fi dabalataa websaayitichaa irratti fayyadamaa jira.
​Fayyadama (Usage)
​Barattootni odeeffannoo barnoota isaanii fi klabbii isaanii (Federalism Club) argachuu danda'u.
​Yaada ykn gaaffii qabdan hunda kuta "Nu Qunnamaa" (Contact Us) jedhu fayyadamuun nutti erguu dandeessu.
